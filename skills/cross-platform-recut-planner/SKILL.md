# Cross Platform Recut Planner

## What this does (plain language)

Turn one content idea into three usable versions:
- TikTok cut
- Reels cut
- Shorts cut

Keep one core message, but adapt hook style, pacing, and CTA for each platform.

## Workflow

### 1) Capture the source unit

Collect:
- core message
- target audience
- desired action (follow, click, comment, buy)
- available footage/assets

### 2) Define platform intent

Set one goal per platform:
- TikTok: discovery + fast resonance
- Reels: social share + visual polish
- Shorts: search + retention consistency

### 3) Build recut matrix

For each platform, produce:
- hook line
- 15–45s beat structure
- caption style
- on-screen text cues
- CTA line

### 4) Output production pack

Return:
- 3 final scripts (TikTok/Reels/Shorts)
- edit notes (cuts, B-roll, subtitle style)
- posting notes (timing and sequencing)

## Quality rules

- Never ship identical scripts across all 3 platforms.
- Keep each version natively worded and behavior-aware.
- Preserve one central promise across all variants.
