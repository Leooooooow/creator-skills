## 4) Micro-outline (10-second flow) for selected hooks

### Top 1 — #7 Proof-first
- **Hook line:** “This AI hook format moved one Shop video from 18% to 34% 3-second hold — here’s the exact opener.”
- **10-second flow:**
  - 0–2s: Deliver hook + show retention numbers on screen
  - 2–6s: Show “before hook” vs “after hook” text
  - 6–10s: Reveal the format: **Problem + Outcome + Product fit**
- **On-screen text:** `18% → 34% in 3-second hold`
- **CTA:** “Tap bio and test this format on your next product video.”

### Top 2 — #5 Mistake/fix
- **Hook line:** “You’re losing viewers by second 2 because you explain too soon — say the result first.”
- **10-second flow:**
  - 0–2s: State the mistake
  - 2–6s: Show bad opener vs fixed opener
  - 6–10s: Generate a fixed line with the AI script tool
- **On-screen text:** `Outcome first > setup first`
- **CTA:** “Comment your current opener — I’ll rewrite one live style.”

### Top 3 — #3 Fast checklist
- **Hook line:** “Before posting, check this: pain, proof, promise — miss one and retention drops.”
- **10-second flow:**
  - 0–2s: Flash the 3-word checklist
  - 2–6s: Map each word to one sample line
  - 6–10s: Combine into one final hook sentence
- **On-screen text:** `Pain + Proof + Promise`
- **CTA:** “Save this checklist and run it before every upload.”

### Backup — #11 Urgent timing
- **Hook line:** “Shop traffic peaks tonight — fix this one line before you post.”
- **10-second flow:**
  - 0–2s: Urgency + timing cue
  - 2–6s: Show one weak first line
  - 6–10s: Replace with stronger line from tool output
- **On-screen text:** `Fix 1 line before posting`
- **CTA:** “Try one rewrite now and post within the next hour.”
