# Smoke Test — creator-monetization-risk-checker

## Scenario Draft
> "Make $10k/day guaranteed from one repost. No effort. This hack always bypasses platform rules."

## 1) Parsed Risk Signals
- **Guaranteed earnings claim** ("$10k/day guaranteed") → high misinformation / scam signal.
- **Effortless outcome claim** ("No effort") → deceptive expectation setting.
- **Policy-evasion language** ("bypasses platform rules") → explicit compliance violation signal.
- **Reuse framing** ("one repost") → originality/reused-content monetization risk.

## 2) Risk Scores (1–5)
- **Policy / advertiser safety:** 5
- **Originality / reuse:** 4
- **Misleading-claim risk:** 5
- **Brand suitability:** 5

**Overall risk score:** **4.75 / 5**  
**Verdict:** 🔴 **RED (major risk; rework before publishing)**

## 3) Yellow/Red Fixes (with safer rewrites)
1. **[RED] Problem line:** "Make $10k/day guaranteed"  
   **Safer rewrite:** "Some creators increase revenue with repurposed formats, but results vary by niche, offer, and execution."  
   **Confidence:** High

2. **[RED] Problem line:** "This hack always bypasses platform rules"  
   **Safer rewrite:** "Use platform-compliant methods: original or licensed assets, plus meaningful transformation."  
   **Confidence:** High

3. **[YELLOW] Problem line:** "from one repost. No effort."  
   **Safer rewrite:** "From one repurposed post idea, test multiple edited versions and improve with data-driven iteration."  
   **Confidence:** Medium-High

## 4) Top 3 Fixes by Impact
1. Remove **guaranteed income** claims.
2. Remove all **rule-bypass / hack** language; state compliance explicitly.
3. Replace **"no effort"** framing with realistic process + testing effort.

## Publishable Rewrite (ready to post)
"Can a repurposed post format generate meaningful revenue? Sometimes—if you do it right.

Here’s a compliant 3-step framework:
1) Use original or properly licensed material.
2) Add real transformation (new hook, edit structure, commentary, CTA).
3) Track retention, CTR, and conversions for 7 days, then iterate.

No guaranteed income—just a repeatable process that improves odds over time."
