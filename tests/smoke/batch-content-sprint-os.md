# Batch Content Sprint OS — Smoke Test

## 1) Sprint constraints
- **Capacity:** 18 hours/week
- **Team:** Solo creator (single owner for all tasks)
- **Output target:** 6 shorts + 2 repurposed clips
- **Primary priority:** Consistency first, quality second
- **Time budget split:** ~16.5h production + 1.5h buffer/recovery

## 2) Sprint backlog (Must / Should / Nice)
### Must ship
- Finalize 6 short-form topics + hooks
- Script all 6 shorts
- Batch record footage for all 6 shorts
- Edit + package all 6 shorts (captions/CTA)
- Create 2 repurposed clips from top-performing/raw long segments
- Schedule/publish and log first-24h performance

### Should ship
- 2 alternate hooks/titles for best 3 shorts
- Comment-reply block after first posts go live
- Reusable edit template cleanup for next week

### Nice to have
- 1 trend-reactive bonus short (only if buffer remains)
- 1 behind-the-scenes clip

## 3) Time-blocked production plan (18h total)
- **Ideation + scripting:** 6h
- **Recording:** 4h
- **Editing + repurposing:** 6h
- **Publishing + review:** 1h
- **Buffer:** 1h

## 4) Burnout guardrails
- **Max deep work/day:** 4h hard cap
- **Low-load day:** Saturday (max 1h, no heavy editing/recording)
- **Buffers:** 30 min revision buffer on Thu + 30 min on Fri
- **Focus rhythm:** 50/10 work-break cycles, max 3 deep blocks/day
- **Fallback content options:**
  - Convert 1 planned short into a quick talking-head tip if shoot overruns
  - Repurpose from existing footage archive if new recording slips

## 5) Day-by-day sprint board
| Day | Hours | Owner | Tasks | Deliverables |
|---|---:|---|---|---|
| Mon | 3h | Solo creator | Content angles, hooks, script shorts #1-2 | 2 scripts ready |
| Tue | 3h | Solo creator | Script shorts #3-6, shot list, setup prep | 4 scripts + shot list |
| Wed | 4h | Solo creator | Batch record shorts #1-6, organize footage | Raw footage complete |
| Thu | 3.5h | Solo creator | Edit shorts #1-3, add captions/CTA, 30m buffer | 3 shorts export-ready |
| Fri | 3.5h | Solo creator | Edit shorts #4-6, make 2 repurposed clips, 30m buffer | 3 shorts + 2 clips ready |
| Sat (low-load) | 1h | Solo creator | Schedule/publish queue, light comments, metrics log | Week shipped + metrics started |

### Delivery checklist
- [ ] 6 shorts exported and scheduled/published
- [ ] 2 repurposed clips exported and scheduled/published
- [ ] Captions + CTA added to every asset
- [ ] First-24h metrics logged for optimization
- [ ] Next-week carryover notes captured
