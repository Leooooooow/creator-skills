# Smoke Test — `ugc-brief-to-script-engine`

## 1) Parsed Brief Essentials
- **Product:** TikTok Shop beauty product (daily brightening serum)
- **Audience:** 18–34 skincare buyers who want visible results without a complex routine
- **Key benefit claims:** brighter-looking skin tone, smoother texture look, lightweight hydration
- **Mandatory points:**
  - On-camera **ingredient proof** (show label + callout)
  - Explicit **20% off CTA**
- **Compliance guardrails:** avoid medical/guaranteed claims; use cosmetic wording only
- **Visual constraints:** creator-shot, handheld, natural bathroom/vanity setup

## 2) Script Pack (3 Variants)

### A) Problem / Solution (20–30s)
**Hook:** “My makeup kept catching on dry patches—this is the one step that fixed it.”

**Scene-by-scene shot notes + lines**
1. **Problem close-up (face, dry patch area):**
   - Line: “By noon, my base looked flaky around my cheeks.”
2. **Product + ingredient proof (tight label shot):**
   - Line: “I switched to this serum—see here: **Niacinamide + Hyaluronic Acid** right on the label.”
3. **Application demo (2–3 drops, pat in):**
   - Line: “It sinks in fast, no sticky layer, so my makeup sits smoother.”
4. **After look + CTA (finished makeup):**
   - Line: “If your skin prep is failing you, start here. It’s **20% off in my TikTok Shop**—tap to grab it.”

---

### B) Demo / Proof (20–30s)
**Hook:** “Proof first: watch what this does in 10 seconds.”

**Scene-by-scene shot notes + lines**
1. **Texture demo on hand (drop + spread):**
   - Line: “Lightweight texture—absorbs super fast.”
2. **Ingredient proof macro (label + finger point):**
   - Line: “Key ingredients: **Niacinamide + Hyaluronic Acid**—you can literally see it on-pack.”
3. **Split-screen before/after finish (before: dull, after: glowy):**
   - Line: “Same lighting, same camera—just better skin prep and glow.”
4. **Shop card on screen + CTA:**
   - Line: “It’s currently **20% off** in my TikTok Shop. Hit the product link now.”

---

### C) Personal Story (25–35s)
**Hook:** “I stopped buying random skincare and kept only this one serum in my routine.”

**Scene-by-scene shot notes + lines**
1. **Creator selfie talking-head (authentic tone):**
   - Line: “I wanted one daily product that made my skin look fresher without heavy layers.”
2. **Ingredient proof shot (label + on-screen text):**
   - Line: “I picked this because of **Niacinamide + Hyaluronic Acid**—clear and easy to verify.”
3. **Routine insert (AM use before sunscreen/makeup):**
   - Line: “I use 2 drops in the morning, then SPF. That’s it.”
4. **Lifestyle payoff shot (outdoor natural light):**
   - Line: “My skin looks smoother and more even on camera now.”
5. **CTA close:**
   - Line: “If you want a simple routine upgrade, it’s **20% off in my TikTok Shop** today.”

## 3) Revision-Safe Options
- **Softer-claim version:**
  - Replace “fixed it” with “helped improve the look of it.”
  - Replace “better skin prep and glow” with “a fresher, more hydrated look.”
- **15s cut (for all 3 scripts):**
  - Hook (3s) → Ingredient proof (4s) → One demo/result shot (5s) → **20% off CTA** (3s)
- **Stronger proof-first cut:**
  - Start on ingredient label macro in first second, then immediate application + same-lighting result + CTA
