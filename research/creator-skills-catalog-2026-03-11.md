# Creator Skills Catalog (Agent + Human Friendly)

Goal: explain what each skill does in plain language so both agents and people can adopt them quickly.

## 1) creator-search-intent-radar
- Human version: Find what your audience is searching for, then turn it into this week’s content plan.
- Agent version: Trigger when asked for trend/search-based topic discovery and weekly backlog ranking.

## 2) short-video-hook-lab
- Human version: Generate stronger openings so people stop scrolling in the first 1–3 seconds.
- Agent version: Trigger when retention is low or user asks for hook rewrites/variants.

## 3) creator-deal-ops
- Human version: Keep brand deals organized, scoped, and paid on time.
- Agent version: Trigger for sponsorship workflow, deliverables, revisions, and payment tracking.

## 4) cross-platform-recut-planner
- Human version: Turn one idea into TikTok/Reels/Shorts versions that actually fit each platform.
- Agent version: Trigger for repurposing one source asset/script into platform-native short cuts.

## 5) creator-monetization-risk-checker
- Human version: Catch risky lines before posting so you don’t lose monetization.
- Agent version: Trigger for pre-publish policy/ad-safety/originality risk triage.

## 6) ugc-brief-to-script-engine
- Human version: Convert a messy brand brief into creator-native UGC scripts fast.
- Agent version: Trigger when user provides sponsor brief and needs script pack + shot list.

## 7) creator-attribution-lite
- Human version: See which content drives real business outcomes, not just vanity metrics.
- Agent version: Trigger for lightweight ROI mapping from content outputs to leads/sales.

## 8) batch-content-sprint-os
- Human version: Plan a realistic weekly content sprint without burnout.
- Agent version: Trigger for weekly planning, batching, and execution scheduling.

## 9) creator-proof-portfolio-builder
- Human version: Build a one-page proof pack that helps brands say yes.
- Agent version: Trigger for media-kit upgrade, case snapshot, and outreach-ready proof.

## 10) comment-to-content-recycler
- Human version: Turn audience comments and DMs into endless relevant content ideas.
- Agent version: Trigger for FAQ/comment mining and intent-cluster content generation.
